# How to use

### Clone the project

#### git clone https://github.com/NBalboa/ga.git

### Go to the project directory

#### cd ga

### Install dependencies

#### npm install

### Build the project

#### npm run build

### Link the project

#### npm link

### run the poject

#### ga `<username>`
